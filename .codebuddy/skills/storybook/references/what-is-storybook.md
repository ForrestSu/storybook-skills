# 儿童绘本设计原则

## 什么是儿童绘本

儿童绘本（Picture Book）是以图画为主、文字为辅的叙事作品。每一页的图画不是文字的"配图"，而是独立承担叙事功能的视觉语言。好的绘本中，图画能讲述文字没有说出的故事。

## 核心设计原则

### 1. 图画即叙事

- 每一页的图画必须能独立传达该页的情节进展
- 文字与图画互补，而非重复——文字说"小熊很害怕"，图画应展示害怕的具体场景，而不是画一个写着"害怕"的表情
- 利用画面构图引导读者的视线和情绪

### 2. 翻页的节奏感

- 绘本的核心叙事单位是"翻页"（page turn）
- 每一次翻页都应带来信息量的变化：悬念、转折、惊喜、或情绪递进
- 避免连续两页传达相同的情绪强度——需要有起伏
- 经典节奏: 铺垫 → 铺垫 → 高潮 → 舒缓 → 结局

### 3. 画面的一致性

- **角色造型一致**：主角在每一页的外貌特征（发型、衣着颜色、体型比例）必须完全一致
- **画风一致**：整本绘本必须使用同一种视觉风格，不能混搭写实和卡通
- **色调一致**：建立统一的色彩体系，用色调变化来表达情绪（暖色→安全、冷色→紧张）
- **构图视角一致**：保持同一场景的空间关系逻辑

### 4. 留白与简洁

- 儿童绘本不是信息密度越高越好
- 每页聚焦一个核心画面和一个核心信息
- 留出足够的视觉呼吸空间
- 背景元素为故事服务，不喧宾夺主

### 5. 情绪弧线

一本完整的绘本应有清晰的情绪弧线：

```
开场（安全/日常）
  → 触发事件（好奇/问题出现）
    → 冒险/尝试（紧张/兴奋递增）
      → 高潮（最紧张/最激动的时刻）
    → 解决（释然/成长）
  → 回归（温暖/满足）
结局（比开场更好的状态）
```

## 绘本的常见页数

| 类型 | 页数 | 适合 |
|------|------|------|
| 迷你绘本 | 8-10 页 | 简单概念、低龄认知 |
| 标准绘本 | 12-16 页 | 完整短故事 |
| 长篇绘本 | 20-32 页 | 复杂情节、多角色 |

## 图文配比建议

- 每页文字不超过 2-3 句话（低龄可以只有 1 句）
- 文字放置位置要考虑与画面的和谐，不遮挡关键画面元素
- 文字区域留出固定的位置（顶部、底部、或侧边），整本保持一致

## 适合 AI 生成的画风推荐

| 画风 | 描述 | 适合内容 |
|------|------|---------|
| 水彩风 (Watercolor) | 柔和、温暖、有手绘质感 | 自然、情感、温馨故事 |
| 扁平插画 (Flat Illustration) | 色块分明、现代感强 | 科普、认知、日常生活 |
| 数字绘画 (Digital Painting) | 精致、色彩丰富 | 冒险、幻想、宏大场景 |
| 蜡笔风 (Crayon) | 童趣、粗犷、有纹理 | 低龄、手工感、亲切 |
| 线条插画 (Line Art) | 干净、简约、有设计感 | 教育、对话、概念解释 |
| 日系绘本 (Japanese Picture Book) | 清新、留白多、意境感 | 哲理、安静、诗意故事 |

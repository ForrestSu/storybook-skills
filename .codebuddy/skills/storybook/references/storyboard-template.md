## 故事创作方法论

### 核心叙事结构："OH DEAR → HURRAY"

经典儿童绘本的叙事结构可以概括为：

1. **日常场景建立（Setup）** — 角色在做某件平常的事，建立安全感
2. **"哎呀！"时刻（Conflict）** — 因为角色自己的行为/失误，出了小问题。注意：冲突来自角色内部（好奇、犯错、害怕），不是外部恶势力
3. **情绪低谷（Emotional Low）** — 角色感到伤心/尴尬/沮丧/害怕，画面明确展现情绪
4. **"太好了！"时刻（Resolution）** — 通过角色自身的成长或朋友的帮助解决问题。关键词：善意、理解、行动——而不是说教
5. **温暖结尾（Warm Ending）** — 状态比开头更好，角色有了成长

### 好故事的 6 条铁律

1. **从不说教**：解决问题的方式是行动和体验，不是大道理。不要让角色说"我们要学会XX"。
2. **不怪罪**：出了问题没有人批评指责，朋友只是默默帮忙。
3. **自然后果**：问题是角色自己行为的自然结果，不是惩罚。例如：不刷牙→牙齿不舒服，不是"牙齿仙子惩罚了他"。
4. **情绪真实**：允许角色哭泣、生气、害怕。不要求角色"勇敢"或"坚强"。
5. **Show Don't Tell**：用画面和动作表达情绪，而不是用文字描述"他很难过"。
6. **友谊/亲情是核心动力**：每个故事的核心都是"有人陪伴真好"。

### 角色弧线设计

每个故事中至少一个角色要有明确的内在变化：

- **认知变化**：从不知道→知道（如：不知道刷牙重要→知道了）
- **情感变化**：从害怕→勇敢，从抗拒→接受（如：害怕水→喜欢游泳）
- **关系变化**：从疏远→亲近，从矛盾→和解

弧线必须是渐进的，不能突变。读者要能跟着角色一起经历这个变化。

### 冲突构建技巧

好的儿童绘本冲突有以下特点：

| 好的冲突 | 差的冲突 |
|---------|---------|
| 角色自己的性格导致的 | 凭空出现的外部威胁 |
| 幼儿真实会遇到的 | 大人世界的问题 |
| 有共鸣的（小读者会想"我也这样"） | 距离太远的 |
| 可以通过成长解决的 | 需要超能力解决的 |
| 一个核心冲突，逐步升级 | 太多冲突堆在一起 |

### 不同年龄段的叙事策略

| 年龄段 | 故事长度 | 冲突类型 | 语言特征 | 重复结构 |
|--------|---------|---------|---------|---------|
| 0-2岁 | 8-10页 | 认知类（这是什么？在哪里？） | 单词/短语为主 | 高度重复，每页类似句式 |
| 2-4岁 | 10-12页 | 日常冲突（不想做XX、弄丢了XX） | 简单句，每页1-2句 | 中度重复，有叠句 |
| 4-6岁 | 12-16页 | 社交冲突（朋友矛盾、害怕新事物） | 稍复杂句，每页2-3句 | 低重复，情节驱动 |

### 常见故事模式

以下是经过验证的故事模式，可以直接使用或变形：

1. **"我不要！"模式**：角色抗拒做某事 → 自然后果来了 → 发现做这件事其实很有趣
   - 例：不想刷牙 → 牙齿不舒服 → 朋友把刷牙变成游戏

2. **"找回来"模式**：失去了重要的东西 → 寻找过程中发现新事物 → 找到了比原来更好的
   - 例：气球飞走了 → 朋友安慰+一起做新的 → 发现友谊比气球更珍贵

3. **"第一次"模式**：角色害怕/不敢做新事物 → 在朋友鼓励下尝试 → 发现没有想象中可怕
   - 例：害怕游泳 → 朋友陪着慢慢来 → 发现水里好好玩

4. **"吵架和好"模式**：两个角色意见不合 → 争吵/冷战 → 发现对方的好/需要对方 → 和好
   - 例：都想当队长 → 分别试了结果都不行 → 一起合作才成功

## 分镜设计原则

### 镜头语言

| 镜头类型 | 用途 | 示例 |
|----------|------|------|
| 全景/远景 | 建立场景、展示环境 | 森林全貌、小镇远景 |
| 中景 | 展示角色互动 | 两个角色面对面说话 |
| 近景 | 强调角色情绪 | 角色的上半身，能看清表情 |
| 特写 | 聚焦关键物品或表情 | 一只手拿起钥匙、眼睛特写 |
| 俯视 | 展示空间关系、营造渺小感 | 角色站在悬崖边向下看 |
| 仰视 | 营造威严、高大感 | 从角色脚下向上看大树 |

### 构图模式

| 构图 | 描述 | 适合场景 |
|------|------|---------|
| 中央构图 | 主体居中 | 角色独白、重要时刻 |
| 三分法 | 主体在三分线交点 | 大多数叙事页面 |
| 对角线 | 主体沿对角线分布 | 运动、冲突、紧张 |
| 框中框 | 通过窗户/门框/树洞看 | 偷窥、发现、进入新世界 |
| 留白构图 | 大面积空白+小主体 | 孤独、安静、广阔 |

### 翻页节奏设计

好的绘本利用"翻页"制造叙事节奏：

```
第 1 页: 建立日常（平静）
第 2 页: 发现异常 / 触发事件（好奇）
第 3 页: 开始行动（兴奋）
第 4-5 页: 遇到困难 / 挑战递增（紧张↑）
第 6 页: 最大困境（高潮前）
第 7 页: 转折 / 灵感出现（翻页惊喜！）
第 8 页: 解决问题（释然）
第 9 页: 回归 / 成长（温暖）
第 10 页: 结尾画面（满足 + 余韵）
```

关键技巧：
- 在左页（偶数页）制造悬念，在右页（奇数页/翻页后）给出答案
- 最重要的转折放在需要翻页才能看到的位置
- 连续两页不要使用相同的镜头距离

## 注意事项

- 每一页的 prompt 中**必须**包含角色外貌的关键描述，即使有参考图
- 明确指定画风和色调，每一页保持一致
- 场景描述要具体到可以直接用于图片生成，避免抽象描述
- 每页的故事文字应直接渲染在画面中（通常在底部或顶部），需在分镜中标注文字内容和位置
